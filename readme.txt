This is a 3D interpretation of the Atari game "Missile Command". To run it, use a version of Anaconda with python 2.7 and VPython installed. The play the game, run the "run.py" file.
Anaconda can be installed here: https://www.anaconda.com/download/ (2.7 version)
VPython can then be downloaded with this command in a bash terminal: conda install -c mwcraig vpython
